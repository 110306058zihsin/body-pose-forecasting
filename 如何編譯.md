# 如何編譯 Project_Proposal.tex

## 方法一：用 Overleaf（不用安裝）

1. 打開 **https://www.overleaf.com**（可免費註冊）
2. 點 **New Project** → **Upload Project**
3. 上傳 `Project_Proposal.tex`（或新建專案後把 .tex 內容貼上）
4. 左側編輯、右側會自動編譯出 PDF，也可點 **Recompile** 手動編譯
5. 編譯完成後點 **PDF** 旁的下載即可得到 PDF

不需要在電腦上安裝任何 LaTeX。

---

## 方法二：本機用指令編譯（需已安裝 TeX）

在終端機進入專案資料夾後執行：

```bash
cd /Users/xin/Desktop/CSE586/project1
pdflatex Project_Proposal.tex
```

若未安裝 TeX：
- **Mac：** 安裝 [MacTeX](https://tug.org/mactex/) 或較精簡的 `brew install --cask basictex`
- **Windows：** 安裝 [MiKTeX](https://miktex.org/) 或 [TeX Live](https://tug.org/texlive/)
