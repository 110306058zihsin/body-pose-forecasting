# CSE 586 Project 1: Body Pose Forecasting

Body pose forecasting using a Transformer in Vposer's 32-D latent space. Predicts 2–5 seconds ahead and is evaluated against zero-velocity and constant-velocity baselines with MPJPE.

**Author:** Tzuhsin Chiu

## Proposal

- `Project_Proposal.tex` — 1-page project proposal (two-column LaTeX).
- Compile: `pdflatex Project_Proposal.tex` or use [Overleaf](https://www.overleaf.com).

## Setup (later)

- Data: AMASS (SMPL); encode/decode with Vposer.
- Training: PyTorch (or similar); planned to run on Google Colab.
